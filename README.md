# 🎯 Sudoku

Um jogo de Sudoku minimalista e elegante, construído com **HTML5**, **CSS3** e **JavaScript puro** — sem frameworks, sem bibliotecas, sem dependências.

✨ **Versão Beta** — disponível para testes e feedback.

---

## 🚀 Jogar Agora

Acesse a versão online:  
https://makaiblue.github.io/Sudoku/
## 🎮 Funcionalidades

- ✔ **Novo jogo** — escolha entre Fácil, Médio e Difícil
- ✔ **Continuar partida** — salvamento automático no navegador
- ✔ **Cronômetro** — acompanhe seu tempo
- ✔ **Tema claro/escuro** — alternância com um clique
- ✔ **Destaques visuais** — linha, coluna, bloco e números iguais
- ✔ **Dica** — revela uma célula aleatória
- ✔ **Apagar número** — limpa a célula selecionada
- ✔ **Verificar vitória** — validação automática ao completar
- ✔ **Animações suaves** — feedback visual ao acertar ou errar
- ✔ **Responsivo** — jogue no celular, tablet ou desktop
- ✔ **Salvamento automático** — nunca perca seu progresso

| Dificuldade | Números visíveis | Ideal para |

## 🛠️ Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — variáveis, grid, animações, temas
- **JavaScript (ES6+)** — lógica pura, módulos IIFE
- **GitHub Pages** — hospedagem gratuita

---
---

## 🧠 Algoritmo

- Geração de tabuleiro completo usando **backtracking**
- Remoção de células garantindo **solução única** (contador de soluções com poda)
- Validação em tempo real (linha, coluna, bloco 3×3)

---

## 📱 Responsividade

O layout se adapta automaticamente a qualquer tamanho de tela, mantendo a proporção do grid e o conforto dos botões.
